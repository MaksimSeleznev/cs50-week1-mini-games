# 🎮 Mini Games in C

A small collection of console games written in C while studying the CS50 course.

This repository is focused on practicing fundamental programming concepts such as:

- loops
- functions
- arrays
- strings
- game logic
- user input handling

---

# 🎮 Games

## 1️⃣ Guess The Number

A simple console game where the player must guess the correct number.

Features:
- random number generation
- attempt based gameplay
- simple console interface

---

## 2️⃣ Hangman

A classic Hangman word guessing game.

Features:

- 4 difficulty levels
- random word selection
- attempt counter
- used letters display
- win/lose conditions

Example gameplay:

Choose level from 1 to 4:
2

[Attempt #1]
_ _ _ _

Enter your letter: a

Used letters: [a]

_ a _ _
